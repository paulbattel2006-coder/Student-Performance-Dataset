# Student Performance Dataset

Gruppenprojekt zum Datensatz `student_performance_data.csv` (Portugiesisch-Noten an zwei portugiesischen Sekundarschulen). Aufgabenstellung siehe [README_Angabe.md](README_Angabe.md), Datensatz-Doku siehe [data/student_performance_data.md](data/student_performance_data.md).

## Setup

Abhängigkeiten installieren:

```bash
pip install -r requirements.txt
```

Notebook starten:

```bash
jupyter notebook notebooks/student_performance.ipynb
```
